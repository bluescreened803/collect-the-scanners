# Collect the Scanners
This game is designed to be a short, interactive activity for a session I ran around timetabling. It is built using [p5.js](https://p5js.org/).

## Controls
Click into the window and move your mouse to control the ID card. Aim for the highest score you can by collecting the falling scanners, while avoiding the brick walls. 

> [!NOTE]
> A mouse or trackpad is required to play, touchscreens are not supported.
