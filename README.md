This game is designed to be a short, interactive activity for a session I ran around timetabling. Built in July 2025 using JavaScript and [p5.js](https://p5js.org/).

## How to Play
- Click into the window and move your mouse to control the ID card. 
- Aim for the highest score you can by collecting the falling scanners. 
- Try to avoid hitting the brick walls as you will lose a life. 

> [!NOTE]
> A mouse or trackpad is required to play, touchscreens are not supported.
